Aprendendo a organizar listas

Dia dia, listas ordenadas:

Vou organizar todas as atividades do meu dia.

Básico

  1. Acoradar
  2. Tomar banho
  3. Escovar os dentes

Add o type dentro de Ol para mudar a formatação da minha lista.
Exemplo:

Usando o type dentro de ol:

  1. café da manhã
    1. Ovos
    2. Pães
    3. Cereais
  2. indo para o trabalho
    1. Separo a camisa
    2. Separo a calça
    3. sapado
    4. me arrumo

Listas não ordenadas

No lugar de "ol" uso o ul dentro de ul posso
usar TYPE = disc, circle ou square.

  1. Indo para o trabalho
    • Saio de casa
    • Chego na parada
    • Pego o autocarro
    • Chego no trabalho
  2. No trabalho
    • Separo lista de bebidas
    • Organizo a montra
    • Limpo a loja
